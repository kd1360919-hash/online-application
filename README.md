オンライン申請

# online-application

HTML/CSS/JavaScript で構築した、住民票のオンライン申請を想定したデモアプリケーションです。  
申請内容の入力、手数料計算、申請番号の自動発行、LocalStorage を利用した申請履歴管理など、  
実務に近い画面構成と処理フローを意識して制作しました。

---

## 📌 主な機能
- 住民票の種類選択（世帯全員 / 本人のみ / 一部）
- 通数入力と手数料の自動計算
- 受取方法・支払方法の選択
- 入力チェック（未入力・不正値の検出）
- 申請番号の自動発行（AP-0001 形式）
- LocalStorage による申請履歴の保存
- 申請一覧の表示
- 全件削除機能

---

## 🛠 使用技術
- **HTML5**
- **CSS3**
- **JavaScript（Vanilla JS）**
- **LocalStorage**

フレームワークは使用せず、基礎技術のみで構築しています。
