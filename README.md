オンライン申請

# online-application

オンライン申請を想定したデモアプリです。
申請入力から履歴管理までをフロントエンドのみで再現しています。

---

## 📌 主な機能
・申請内容入力フォーム
・手数料の自動計算
・入力バリデーション
・申請番号自動発行（AP-0001形式）
・申請履歴の保存・表示
・全件削除機能

---

## 🛠 使用技術
- **HTML5**
- **CSS3**
- **JavaScript（Vanilla JS）**
- **LocalStorage**

フレームワークは使用せず、基礎技術のみで構築しています。

---

## 💡 制作のポイント
- 実際の自治体サイトを参考にしつつ、学習用として簡潔な UI に調整
- 入力チェックや手数料計算など、業務システムで求められる処理を意識
- LocalStorage を利用し、バックエンドなしで履歴管理を実現
- VSCode を使用して開発

---

## ⚠ 注意
本アプリは学習目的のデモであり、実際の申請には使用できません。
